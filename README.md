# Wave Interference

ブラウザで動作する、3D波干渉シミュレーターです。  
`HTML / CSS / JavaScript` のみで実装しており、ビルド不要で動かせます。

## 主な機能

- 3D水面上で複数波の干渉を可視化
- 波の追加・削除
- 各波のパラメータ調整
  - 振幅
  - 波長
  - 速度
  - 位相
  - 発生源 X / Z
- 発生源マーカーのドラッグ移動（3Dビュー上）
- 任意位置・任意角度での断面観測
- 断面グラフ表示
  - 合成波（黒線）
  - 各波の寄与（色線）
  - 発生源位置マーカー
  - 観測点マーカー
- プリセット保存 / 読込 / 削除（`localStorage`）

## 使い方

1. `index.html` をブラウザで開く
2. サイドメニューで波を追加・調整する
3. 3D水面をクリックして観測位置を指定する
4. 断面角度を変えて、断面グラフを確認する

## ファイル構成

- `index.html`  
  画面レイアウト、CDN読み込み、UI要素
- `style.css`  
  全体デザイン、レスポンシブ、スライダーやカードのスタイル
- `script.js`  
  Three.js描画、波動計算、UIイベント、断面グラフ描画
- `logo.svg`  
  タイトル横アイコン / favicon 用ロゴ

## 技術スタック

- [Three.js](https://threejs.org/)（CDN）
- Vanilla JavaScript
- HTML / CSS

## 注意事項

- プリセットはブラウザの `localStorage` に保存されます。
- ブラウザキャッシュにより見た目変更が反映されない場合は、ハードリロード（`Ctrl + F5`）を試してください。
